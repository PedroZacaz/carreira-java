# Carreira Java - Alura

Este repositório contém meus estudos, anotações e projetos desenvolvidos durante a **Formação Java** da [Alura](https://www.alura.com.br/cursos-online-programacao/java). O objetivo é registrar minha evolução desde a lógica de programação até o desenvolvimento de APIs robustas.

---

## Tecnologias e Ferramentas
* **Linguagem:** Java 21
* **IDE:** IntelliJ IDEA / Eclipse
* **Banco de Dados:** MySQL / PostgreSQL
* **Persistência:** JDBC / JPA / Hibernate
* **Framework principal:** Spring Boot (em breve)

---

## Conteúdo da Formação

Abaixo, os módulos que compõem minha jornada de aprendizado:

- [x] **Java JRE e JDK:** Compilação, variáveis e controle de fluxo.
- [x] **Java OO:** Introdução à Orientação a Objetos (Classes, Atributos, Métodos).
- [ ] **Java Polimorfismo:** Herança e Interfaces.
- [ ] **Java Exceções:** Aprendendo a criar e tratar erros.
- [ ] **JDBC:** Persistência de dados com bancos relacionais.
- [ ] **Spring Boot:** Criação de APIs REST.
